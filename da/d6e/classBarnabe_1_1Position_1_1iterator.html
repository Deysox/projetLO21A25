Description détaillée

Itérateur sur les positions voisines d'une case donnée. L'opérateur * renvoie un objet de type Position. Le parcours s'effectue dans le sens horaire, dans le sens des aiguilles d'une montre à partir de la rotation 0 (case (x;y+1)).
